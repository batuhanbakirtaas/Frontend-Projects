### **Frontend Small Projects** 🚀

Welcome to the **Frontend Small Projects** folder! This section contains a variety of small-scale projects designed to practice and enhance frontend development skills.  

---

#### **Tech Stack** 🛠️  
The projects in this folder utilize the following technologies:
- **HTML**: Building the structure of web pages.  
- **CSS**: Styling for visually engaging designs.  
- **Bootstrap**: Creating responsive and mobile-first designs.  
- **JavaScript**: Adding interactivity and functionality.  
- **React**: Developing modern, component-based user interfaces.  

---

#### **Purpose** 🎯  
This folder is dedicated to:  
- Practicing key frontend concepts and techniques.  
- Exploring and experimenting with modern web technologies.  
- Building smaller applications that demonstrate specific skills or ideas.  

---
